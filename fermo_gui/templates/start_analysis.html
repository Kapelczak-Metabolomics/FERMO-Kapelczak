Heads Up Online Users!

Due to computational restrains, the online version of FERMO has a few restrictions in place:

  • File sizes: the maximum accepted size of file uploads is 7 MB.

  • Feature table: the maximum allowed number of molecular features is 1000.

  • Job run time: the maximum allowed run time for a single job is 60 minutes, after which the job terminates prematurely.

  • Module run time: the maximum allowed run time for a single module is 10 minutes. If a module terminates due to a timeout, this is written to the summary.txt file.

  • Job storage: all analysis jobs are stored for 4 weeks and are then deleted. Make sure to download your results!

If you want to bypass these restrictions, you can run FERMO offline or use the fermo_core command line tool.
